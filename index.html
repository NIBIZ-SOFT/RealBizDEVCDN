🔍 SmartSelect

Lightweight Searchable Dropdown System - Auto-initializes on elements with class "smart-select"

📦 CDN Links

Add these to your HTML head section:

<!-- SmartSelect CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/NIBIZ-SOFT/RealBizDEVCDN/SmartSelect.css">

<!-- SmartSelect JS (add before closing body tag) -->
<script src="https://cdn.jsdelivr.net/gh/NIBIZ-SOFT/RealBizDEVCDN/SmartSelect.js"></script>

Or use local files:

<link rel="stylesheet" href="./SmartSelect.css">
<script src="./SmartSelect.js"></script>

🚀 Basic Usage

Simply add the class smart-select to any select element:

<select class="smart-select form-select">
  <option value="">Select an option...</option>
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
</select>

✨ Features

  • ✅ Auto-initialization - Just add class "smart-select"
  • ✅ Search functionality - Type to filter options
  • ✅ Keyboard navigation - Arrow keys, Enter, Escape
  • ✅ Lightweight - No dependencies (except vanilla JS)
  • ✅ Bootstrap compatible - Works with Bootstrap forms
  • ✅ Responsive - Works on all screen sizes
  • ✅ Accessible - Keyboard and mouse support

💻 Programmatic Usage

You can also initialize SmartSelect programmatically:

// Auto-initialize all selects with class "smart-select"
// This happens automatically, but you can trigger it manually:
initSmartSelect();

// Or create instance manually:
var select = document.querySelector("#mySelect");
var smartSelect = new SmartSelect(select, {
  placeholder: "Custom placeholder..."
});
